Greenfoot Game

Game ini dibuat menggunakan Greenfoot dengan gameplay yang sederhana namun menantang. Pemain mengendalikan ambulance sebagai karakter utama, menghindari mobil hijau dan biru sebagai rintangan, serta mengumpulkan medkit untuk mencapai skor kemenangan.

Game dalam Keadaan Menang

Ambulance berhasil mengumpulkan medkit dan menyelamatkan pasien.

Game dalam Keadaan Kalah

Ambulance bertabrakan dengan mobil hijau atau biru, menyebabkan game over.
